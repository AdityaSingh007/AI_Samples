# CAE AI Model Comparison Assistant - Startup Checklist

Follow these steps before running the application:

1. Ensure the GitHub Copilot CLI executable is present at:
   `C:\Users\{your_sesa_id}\AppData\Roaming\vendor\copilot\copilot.exe`

2. If the CLI is not installed, download it from:
   `https://github.com/github/copilot-cli/releases/tag/v0.0.410`

3. Set the GitHub Copilot token in the system environment variable named:
   `GITHUB_TOKEN`

4. Ensure the CAE API is running locally at:
   `https://localhost:5001`

5. Have valid CAE API credentials available for authentication when prompted.

6. Ensure the source and target folders exist at:
   `C:\Users\{your_sesa_id}\AppData\Roaming\cae_import_project`

7. Ensure the source and target exported project files are present in the above folders.
